# Installation

First, you need to install the CLI:

```shell
$ npm install -g @luxaurajs/cli
```

Afterwards, clone the base repository:

```shell
$ git clone https://github.com/luxaurajs/base
```

Build your site:
```shell
$ luxaura build
```

Serve your site:
```shell
$ luxaura serve
```

Your site will be available at <http://localhost:5000>.
